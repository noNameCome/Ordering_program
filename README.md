# 발주 확인 프로그램 사용 설명서

> 처음 쓰시는 분도 이 설명서만 보면 바로 사용할 수 있습니다.
> 문의: 정현철 대표 (boardvga) / 안세현 사원 (ash)

---

## 📌 이 프로그램이 뭔가요?

부품/물품 **발주(주문) 현황을 회사 전체가 함께 실시간으로 보고 관리**하는 프로그램입니다.

- 누가 무엇을 주문했는지 한눈에 확인
- 발주 → 입고까지 상태를 단계별로 관리
- 메모로 담당자끼리 소통
- NAS 서버에 자동 저장 → 사무실 모든 PC에서 동시에 확인 가능

## 1. 처음 설치할 때

### 받은 파일 확인

관리자에게 받은 압축 파일을 풀면 아래처럼 생겨있어야 합니다.

```
📁 발주확인프로그램 폴더
  ├── 발주확인프로그램.exe   ← 이걸 실행하세요
  ├── nas_config.json        ← 건드리지 마세요 (NAS 연결 정보)
  └── _internal 폴더         ← 건드리지 마세요 (프로그램 필수 파일)
```

> ⚠️ **주의**: `nas_config.json` 파일과 `_internal` 폴더가 반드시 `발주확인프로그램.exe` 와 **같은 폴더**에 있어야 합니다. 혼자 exe만 옮기면 안 됩니다.

### Windows 보안 경고가 뜰 때

처음 실행하면 이런 창이 뜰 수 있습니다.

```
Windows에서 PC를 보호했습니다
```

→ **"추가 정보"** 클릭 → **"실행"** 클릭하면 됩니다.
→ 한 번만 하면 그 다음부터는 안 뜹니다.

## 2. 프로그램 실행 & 로그인

### 처음 실행 시

`발주확인프로그램.exe` 를 더블클릭하면 **NAS 계정 입력창**이 뜹니다.

| 항목 | 입력할 내용 |
|------|------------|
| NAS 아이디 | 관리자에게 받은 계정 ID (예: `kelly5409`) |
| NAS 비밀번호 | 해당 계정의 비밀번호 |

**저장** 버튼을 누르면 로그인됩니다.

> ✅ 계정 정보는 이 PC에 저장되어, 다음에 실행할 때는 자동으로 로그인됩니다.

### 계정을 바꾸고 싶을 때

메뉴바 **설정 → NAS 계정 변경** 에서 변경할 수 있습니다.

> ❓ 계정을 모르면 **관리자(정현철 대표 / 안세현 사원)**에게 문의하세요.

## 3. 메인 화면 설명

프로그램을 실행하면 아래와 같은 화면이 보입니다.

```
┌────────────────────────────────────────────────────────────┐
│  ➕발주추가  ✏️수정  🗑️삭제  [발주완료] [입고완료] [취소]  │  ← 도구모음
│                             [반품요청] [반품완료] [📋기록열람]│
├────────────────────────────────────────────────────────────┤
│  카테고리 필터   상태 ▼   검색창 🔍   [초기화]             │  ← 필터바
├────────────────────────────────────────────────────────────┤
│  No. │ 카테고리 │  품목명  │ 수량 │ 상태   │ 등록자 │ 발주자│  ← 목록
│   1  │ CPU      │ i9-14900K│  2   │ 대기중 │ 정현철 │       │
│   2  │ 메모리   │ DDR5 32GB│  4   │ 발주완료│ 이하림│ 이하림│
│  ...                                                        │
├────────────────────────────────────────────────────────────┤
│  전체 15건  /  표시 15건    📝 메모 3개    ● 연결됨  [🔄갱신]│  ← 상태바
└────────────────────────────────────────────────────────────┘
```

### 각 부분 설명

| 영역 | 설명 |
|------|------|
| 도구모음 | 발주 추가, 수정, 삭제, 상태 변경 버튼들 |
| 필터바 | 카테고리/상태/검색어로 목록 좁혀보기 |
| 목록 | 현재 진행 중인 발주 전체 목록 |
| 상태바 | 총 건수, 메모 개수, NAS 연결 상태 |

### 자동 새로고침

다른 사람이 발주를 추가하거나 상태를 바꾸면 **약 2.5초마다 자동으로 화면이 갱신**됩니다.
오른쪽 하단에 팝업 알림도 뜹니다.

## 4. 발주 추가하기

새로운 물품을 발주할 때 사용합니다.

### 방법

1. **➕ 발주 추가** 버튼 클릭 (또는 키보드 `Ctrl+N`)
2. 항목을 입력합니다

| 입력 항목 | 설명 | 필수 여부 |
|-----------|------|----------|
| 카테고리 | 품목 분류 (CPU, 메모리 등) | 필수 |
| 품목명 | 주문할 물품 이름 | 필수 |
| 수량 | 주문할 개수 (숫자만 입력) | 필수 |

3. **추가** 버튼 클릭 → 자동 저장

> ✅ 추가한 발주는 사무실 모든 PC에서 바로 확인됩니다.

---

## 5. 발주 수정하기

품목명, 수량, 카테고리를 잘못 입력했을 때 수정합니다.

### 방법

1. 수정할 항목을 **클릭하여 선택** (파란 줄로 강조됨)
2. **✏️ 수정** 버튼 클릭
3. 내용 수정 후 **저장** 버튼 클릭

> ℹ️ 수정 창에서는 메모를 변경할 수 없습니다.
> 메모는 항목을 **더블클릭** → 세부 정보 창에서 추가/삭제합니다.

---

## 6. 상태 변경하기

발주 진행 상황에 따라 상태를 바꿉니다.

### 상태 흐름

```
대기중  →  발주완료  →  입고완료  →  (24시간 후 기록 보관)
               ↓
           반품 요청  →  반품 완료  →  (24시간 후 기록 보관)
               ↓
             취소  →  (24시간 후 기록 보관)
```

### 방법 1 — 도구모음 버튼 사용

1. 변경할 항목 클릭 (여러 개 선택 시 `Ctrl+클릭`)
2. 도구모음에서 원하는 상태 버튼 클릭

### 방법 2 — 우클릭 메뉴 사용

1. 항목에서 마우스 **우클릭**
2. 원하는 상태 선택

> ⚠️ **입고완료 / 반품 완료 / 취소** 상태로 변경한 후 **24시간이 지나면 자동으로 기록 보관**됩니다.
> 보관된 항목은 **📋 기록 열람**에서 다시 확인할 수 있습니다.

---

## 7. 발주 삭제하기

목록에서 발주 항목을 제거합니다.

### 방법

1. 삭제할 항목 클릭 (여러 개 선택 가능)
2. **🗑️ 삭제** 버튼 클릭 (또는 키보드 `Delete`)
3. "삭제하시겠습니까?" 확인창에서 **예** 클릭

> ℹ️ 삭제해도 **기록 보관함에 남아있어** 나중에 📋 기록 열람에서 확인할 수 있습니다.

---

## 8. 메모 달기

발주 항목에 담당자끼리 메시지를 남길 때 사용합니다.

### 메모 추가하기

1. 메모를 달 항목을 **더블클릭**
2. 아래쪽 메모 입력란에 내용 입력
3. **메모 추가** 버튼 클릭

→ 내 이름과 작성 시간이 자동으로 기록됩니다.

```
예시:
[이하림 부장]  납기일 3월 15일로 확인됨       2026-03-09 10:30   [메모 삭제]
[안세현 사원]  대체 모델 가능 — 대표님 확인요  2026-03-09 11:00   [메모 삭제]
```

### 메모 삭제하기

- 해당 메모 옆의 **[메모 삭제]** 버튼 클릭

### 메모 있는 발주 한눈에 보기

- 하단 상태바의 **📝 메모 N개** 를 클릭
- 메모가 달린 발주 목록 팝업 → 더블클릭 시 해당 발주 세부 정보로 이동

> ✅ 메모가 달린 항목은 목록에서 **굵은 글씨**로 표시됩니다.

---

## 9. 검색 & 필터

원하는 발주만 골라서 볼 때 사용합니다.

### 카테고리로 필터

- 화면 왼쪽 카테고리 목록에서 원하는 항목 클릭
- 전체 보려면 맨 위 **[전체]** 클릭

### 상태로 필터

- 상단 **상태 ▼** 드롭다운에서 원하는 상태 선택
- 예: "대기중"만 보기, "발주완료"만 보기

### 키워드 검색

- 검색창에 품목명이나 메모 내용 입력 → 실시간으로 필터링됨

### 정렬하기

- 표 상단의 **열 제목 클릭** → 해당 열 기준으로 정렬
- 한 번 더 클릭하면 반대 방향으로 정렬

### 전부 초기화하기

- **[초기화]** 버튼 클릭 → 모든 필터 해제, 전체 목록 표시

## 10. 엑셀 파일로 한꺼번에 불러오기

여러 품목을 한꺼번에 등록할 때 사용합니다.

### 엑셀 파일 형식

아래처럼 엑셀 파일을 만들면 됩니다.

| A열 (품목명) | B열 (수량) | C열 (카테고리) | D열 (메모) |
|-------------|-----------|--------------|-----------|
| i9-14900K   | 2         | CPU          | 급함      |
| DDR5 32GB   | 4         | 메모리       |           |
| RTX 4080    | 1         | 그래픽카드   |           |

> - **A열(품목명)**, **B열(수량)**은 필수입니다.
> - C열(카테고리), D열(메모)은 없어도 됩니다.
> - 맨 위에 "품목명, 수량, 카테고리, 메모" 같은 **헤더 행이 있어도** 자동으로 건너뜁니다.
> - 헤더 위에 안내 문구 같은 글이 있어도 괜찮습니다. 자동으로 인식합니다.

### 가져오는 방법

1. **➕ 발주 추가** 클릭
2. 추가 창 하단의 **📂 엑셀 파일로 가져오기** 클릭
3. 엑셀 파일 선택
4. 목록 확인 후 **등록** 클릭

## 11. 기록 열람 (처리 완료된 발주 조회)

입고 완료, 취소, 삭제된 발주의 기록을 볼 수 있습니다.

### 여는 방법

도구모음의 **📋 기록 열람** 버튼 클릭

### 기능 설명

| 기능 | 사용법 |
|------|--------|
| 기간 필터 | 오늘 / 이번 주 / 이번 달 / 지난 달 선택 |
| 날짜 검색 | 날짜 입력란에 `2026-03` 입력 → 해당 월 기록 표시 |
| 카테고리 필터 | 드롭다운에서 카테고리 선택 |
| 키워드 검색 | 품목명, 메모, 등록자 이름으로 검색 |
| 상세 정보 보기 | 항목 **더블클릭** → 발주 상세 정보 팝업 (누가, 언제 처리했는지 확인) |
| 최신순 정렬 | 열 제목 클릭으로 정렬 변경 |

> ℹ️ 기록 열람은 **조회 전용**입니다. 여기서 발주를 수정하거나 되살릴 수는 없습니다.

### 관리자 전용 기능

| 기능 | 설명 |
|------|------|
| 선택 삭제 | 선택한 기록을 완전히 삭제 (복구 불가) |
| 기간 일괄 삭제 | N개월 이전 기록 한꺼번에 삭제 |

## 12. 자동 업데이트

프로그램을 실행할 때 **새 버전이 있으면 자동으로 알려줍니다**.

### 업데이트 알림이 뜰 때

```
┌─────────────────────────────┐
│  새 버전이 있습니다!         │
│                             │
│  현재 버전: v1.9.3          │
│  새  버전: v1.9.4           │
│                             │
│  ━━ 업데이트 내용 ━━        │
│  (변경사항이 여기 표시됩니다)│
│                             │
│  지금 업데이트하시겠습니까?  │
│  [예]          [아니오]     │
└─────────────────────────────┘
```

- **예** 클릭 → 자동 다운로드 → 자동 설치 → 자동 재시작
- **아니오** 클릭 → 이번에는 건너뜀 (다음 실행 시엔 안 물어봄)

> ✅ **아무것도 안 해도 됩니다.** "예"만 누르면 끝!
> 업데이트 중에는 "재시작 중..." 화면이 잠깐 표시됩니다.

### 수동으로 업데이트 확인하기

메뉴바 **업데이트 → 업데이트 확인** 클릭

## 13. 자주 묻는 질문 (FAQ)

### Q. 처음 실행했는데 아무것도 안 나와요
**A.** NAS 서버에 연결 중입니다. 잠시 기다려 주세요.
연결이 안 되면 사무실 인터넷이 연결되어 있는지 확인하고, 관리자에게 문의하세요.

---

### Q. "Windows에서 PC를 보호했습니다" 창이 떠요
**A.** **"추가 정보"** → **"실행"** 을 클릭하세요. 처음 한 번만 이렇게 하면 됩니다.

---

### Q. 다른 사람이 추가한 발주가 안 보여요
**A.** 프로그램은 2.5초마다 자동 갱신됩니다. 그래도 안 보이면 오른쪽 하단 **🔄 갱신** 버튼을 눌러보세요.

---

### Q. 실수로 상태를 잘못 바꿨어요
**A.** 항목을 선택하고 올바른 상태 버튼을 다시 클릭하면 됩니다.
단, **입고완료 / 반품 완료 / 취소** 상태로 변경 후 **24시간이 지나면** 자동 보관되어 되돌릴 수 없습니다. 빨리 수정하세요.

---

### Q. 내 이름이 "알 수 없음" 으로 나와요
**A.** 내 NAS 아이디가 직원 목록에 등록되지 않은 것입니다. 관리자에게 직원 등록을 요청하세요.

---

### Q. 발주를 실수로 삭제했어요. 되살릴 수 있나요?
**A.** **📋 기록 열람**에서 삭제된 항목을 조회할 수 있습니다. 내용을 확인해서 다시 추가할 수 있습니다.

---

### Q. 엑셀 파일 가져오기가 안 돼요
**A.** 파일 형식이 `.xlsx` 인지 확인하세요. `.xls` 형식은 지원되지 않습니다.
A열에 품목명, B열에 수량이 있어야 합니다.

---

### Q. 프로그램이 느리거나 화면이 멈춰요
**A.** NAS 서버와 연결이 불안정한 경우 발생합니다.
사무실 인터넷 연결 상태를 확인하고, 잠시 기다린 후 **🔄 갱신** 버튼을 눌러보세요.

---

### Q. 업데이트하면 내 설정이 초기화되나요?
**A.** 아닙니다. 로그인 정보와 설정은 이 PC에 별도로 저장되어 업데이트 후에도 유지됩니다.

---

### Q. nas_config.json 파일을 실수로 삭제했어요
**A.** 관리자에게 연락해서 파일을 다시 받으세요. 이 파일이 없으면 프로그램이 NAS에 연결할 수 없습니다.

## 14. 단축키 모음

| 단축키 | 기능 |
|--------|------|
| `Ctrl + N` | 발주 추가 창 열기 |

| `Delete` | 선택한 항목 삭제 |

| `열 제목 클릭` | 해당 열 기준 정렬 |

## 15. 상태별 색상 의미

| 색상 | 상태 | 의미 |
|------|------|------|
| ⬜ 흰색 | **대기중** | 발주 요청만 된 상태. 아직 실제 주문은 안 됨 |
| 🟡 노란색 | **발주완료** | 실제로 발주(주문)가 완료된 상태 |
| 🟢 초록색 | **입고완료** | 물품이 도착해서 입고된 상태 → 24시간 후 자동 보관 |
| 🟠 주황색 | **반품 요청** | 반품을 접수한 상태 |
| 🩷 분홍색 | **반품 완료** | 반품 처리가 완료된 상태 → 24시간 후 자동 보관 |
| ⬜ 회색 | **취소** | 발주가 취소된 상태 → 24시간 후 자동 보관 |

> ℹ️ 자동 보관된 항목은 **📋 기록 열람**에서 확인할 수 있습니다.
