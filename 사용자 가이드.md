## 1. 설치 및 최초 실행

### 필요 파일 (두 파일이 같은 폴더에 있어야 합니다)
```
📁 배포 폴더
  ├── 발주확인프로그램.exe   ← 실행 파일
  └── nas_config.json        ← NAS 연결 정보 (건드리지 마세요)
```

### 실행 방법
- `발주확인프로그램.exe` 를 더블클릭하여 실행합니다.
- Windows 보안 경고가 뜨면 **"추가 정보" → "실행"** 을 클릭합니다.

> ⚠️ `nas_config.json` 파일이 없으면 NAS에 연결할 수 없습니다.
> 파일이 없다면 관리자(정현철 대표 / 안세현 사원)에게 문의하세요.

## 2. 로그인 (NAS 계정 입력)

프로그램 최초 실행 시 NAS 계정 입력창이 뜹니다.

| 항목 | 내용 |
|------|------|
| NAS 아이디 | 회사 NAS 계정 ID (예: `kelly5409`) |
| NAS 비밀번호 | NAS 접속 비밀번호 |

- 입력 후 **저장** 버튼을 누르면 자동으로 로그인됩니다.
- 계정 정보는 PC에 저장되어 다음 실행 시 자동 로그인됩니다.
- 계정 변경은 **설정 → NAS 계정 변경** 에서 가능합니다.

## 4. 발주 추가

1. **➕ 발주 추가** 버튼 클릭 (또는 `Ctrl+N`)
2. 입력창에서 항목 입력:
   - **카테고리**: 품목 분류 선택
   - **품목명**: 주문할 물품 이름 (필수)
   - **수량**: 주문 수량 (필수, 숫자만)
3. **추가** 버튼 클릭 → NAS에 자동 저장

### 엑셀로 일괄 추가
- 발주 추가 창 하단의 **📂 엑셀 파일로 가져오기** 버튼 클릭
- 아래 형식의 엑셀 파일을 선택합니다:

| A열 (품목명) | B열 (수량) | C열 (카테고리) | D열 (메모) |
|-------------|-----------|--------------|-----------|
| i9-14900K   | 2         | CPU          | 긴급 필요  |
| DDR5 32GB   | 4         | 메모리       |           |

> 첫 번째 행은 헤더(항목명, 수량 등)로 자동 인식하여 건너뜁니다.

---

## 5. 발주 수정

1. 수정할 항목을 **클릭하여 선택**
2. **✏️ 수정** 버튼 클릭 (또는 항목 선택 후 `F2`)
3. 카테고리 / 품목명 / 수량 수정 후 **저장**

> ℹ️ 수정 창에서는 **메모를 변경할 수 없습니다**.
> 메모는 항목을 **더블클릭**하여 세부 정보 창에서 추가/삭제합니다.

---

## 6. 발주 상태 변경

### 방법 1: 도구모음 버튼
- 항목 선택 후 **발주완료 / 입고완료 / 취소** 버튼 클릭
- **여러 항목을 한꺼번에** 선택하여 일괄 변경 가능
  (Ctrl+클릭 또는 Shift+클릭으로 다중 선택)

### 방법 2: 우클릭 메뉴
- 항목에서 마우스 **우클릭** → 원하는 상태 선택

### 상태 흐름
```
대기중 → 발주완료 → 입고완료 (자동 보관 처리)
       → 반품 요청 → 반품 완료 (자동 보관 처리)
       → 취소 (자동 보관 처리)
```

> ⚠️ **입고완료 / 반품 완료 / 취소** 상태는 24시간 후 자동으로 기록 보관됩니다.
> 보관된 항목은 **📋 기록 열람**에서 확인할 수 있습니다.

---

## 7. 발주 삭제

1. 삭제할 항목 선택 (다중 선택 가능)
2. **🗑️ 삭제** 버튼 클릭 (또는 `Delete` 키)
3. 확인 메시지에서 **예** 선택

> ℹ️ 삭제된 항목은 기록 보관함으로 이동되어 나중에 조회할 수 있습니다.

---

## 8. 메모 기능

### 메모 추가
1. 메모를 추가할 항목을 **더블클릭**
2. 세부 정보 창 하단의 메모 입력란에 내용 입력
3. **메모 추가** 버튼 클릭 → 내 이름과 시간이 자동 기록

```
[김보영 사원]  납기일 확인 필요  [메모 삭제]  2026-03-06 14:30
[이하림 부장]  대체품 가능       [메모 삭제]  2026-03-06 15:00
```

### 메모 삭제
- 각 메모 행의 **[메모 삭제]** 버튼 클릭

### 메모 목록 보기
- 하단 상태바의 **📝 메모 N개** 를 클릭하면 메모가 있는 발주 목록을 볼 수 있습니다
- 목록에서 항목을 **더블클릭**하면 해당 발주의 세부 정보로 이동합니다

## 9. 검색 및 필터

### 카테고리 필터
- 왼쪽 카테고리 목록에서 클릭하면 해당 카테고리만 표시

### 상태 필터
- 상단 드롭다운에서 상태 선택 (전체 / 대기중 / 발주완료 / ...)

### 키워드 검색
- 검색창에 품목명을 입력하면 실시간으로 필터링

### 정렬
- 각 열 제목을 클릭하면 해당 열 기준으로 오름차순/내림차순 정렬

### 초기화
- **[초기화]** 버튼 클릭 → 모든 필터/검색 초기화

### 전체 선택
- `Ctrl+A` → 현재 표시된 모든 항목 선택

## 10. 엑셀 가져오기

**발주 추가 창 → 📂 엑셀 파일로 가져오기**

### 엑셀 파일 형식
| A열 | B열 | C열 | D열 |
|-----|-----|-----|-----|
| 품목명 | 수량 | 카테고리 | 메모 |
| i9-14900K | 2 | CPU | 긴급 |

- C열(카테고리), D열(메모)은 생략 가능
- 첫 줄이 헤더면 자동으로 건너뜀
- 가져온 메모는 현재 로그인 사용자 이름으로 자동 기록됨

---

## 11. 기록 열람

**📋 기록 열람** 버튼 클릭

- 입고완료/취소/삭제된 항목들의 기록을 조회할 수 있습니다
- **기간 / 카테고리 / 상태** 로 필터링 가능
- 검색창에서 품목명, 메모, 작성자 통합 검색

### 관리자 전용 기능
- **선택 삭제**: 선택한 기록을 영구 삭제
- **기간별 삭제**: N개월 이전 기록 일괄 삭제

---

## 12. 직원 관리 (관리자 전용)

**설정 → 직원 관리** (관리자만 접근 가능)

### 직원 추가
1. NAS 아이디 입력 (로그인에 사용하는 ID)
2. 이름 입력
3. 직급 입력 (예: 대표, 부장, 실장, 과장, 팀장, 이사, 사원)
4. **추가** 버튼 클릭
5. **💾 NAS에 저장** 버튼 클릭 → 모든 PC에 즉시 반영

### 직원 삭제
- 직원 행 옆의 **삭제** 버튼 클릭 후 **💾 NAS에 저장**

> ℹ️ 저장 버튼을 누르지 않으면 변경사항이 반영되지 않습니다.

## 14. 자주 묻는 질문 (FAQ)

### Q. 프로그램이 느리거나 응답하지 않아요
- NAS 서버가 오프라인이거나 네트워크가 불안정한 경우 발생합니다.
- 사무실 네트워크에 연결되어 있는지 확인하세요.
- 잠시 기다린 후 **🔄 갱신** 버튼을 눌러보세요.

### Q. "NAS 연결 실패" 메시지가 뜨면?
- 인터넷/사무실 네트워크 연결 상태를 확인하세요.
- `nas_config.json` 파일이 실행 파일과 같은 폴더에 있는지 확인하세요.
- 계속 문제가 생기면 관리자에게 문의하세요.

### Q. 다른 사람이 추가한 발주가 안 보여요
- 오른쪽 하단 **🔄 갱신** 버튼을 클릭하세요.
- 프로그램은 2.5초마다 자동으로 새로고침합니다.

### Q. 실수로 상태를 바꿨어요
- 항목을 선택하고 원하는 상태 버튼을 다시 클릭하면 됩니다.
- **입고완료/반품 완료/취소** 상태는 24시간이 지나면 자동 보관됩니다. 그 전에 되돌리세요.

### Q. 엑셀 파일 가져오기가 안 돼요
- 파일 형식이 `.xlsx` 인지 확인하세요 (`.xls` 는 지원 안 됨)
- A열에 품목명, B열에 수량이 있는지 확인하세요.

### Q. 메모를 잘못 추가했어요
- 세부 정보 창(더블클릭)에서 해당 메모 행의 **[메모 삭제]** 버튼을 클릭하세요.

### Q. 내 이름이 "알 수 없음" 으로 나와요
- NAS 아이디가 직원 목록에 등록되지 않은 것입니다.
- 관리자에게 직원 등록을 요청하세요.

### Q. 프로그램을 업데이트하면 설정이 사라지나요?
- 아니요. 설정은 PC에 별도로 저장되어 프로그램을 교체해도 유지됩니다.
